# Project - TimeMachine Database

## Project Origin and Context
- Project retrieved from my GitHub account created with my student email, due to loss of access.

## Description
Database for **Tidsmaskinen**, a company handling timing for sports events. 

Tracks:
- Clubs (sports associations)  
- Events (competitions)  
- Participants and start numbers  
- Results by age group and gender  

## Technologies
- MySQL / MariaDB  
- Java (JDBC)  
- Maven
